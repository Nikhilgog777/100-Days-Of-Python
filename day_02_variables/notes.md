# Notes : Create variable and print
## Problem Statement
– Create variables for your age, height, and favorite color. Print them in a sentence.

### Solution 
age = 20 <br>
height = 1.75 <br>
color = "Blue"

print(f"My age is {age} years old. \nMy height is {height} metres.\nMy favourite color is {color}.")

- create a variable for age, height and color & assign respective values
- print them using f strings
- I used \n as an escaping character

### Output
My age is 20 years old. <br>
My height is 1.75 metres.<br>
My favourite color is Blue.
