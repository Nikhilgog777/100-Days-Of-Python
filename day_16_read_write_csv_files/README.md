# Problem : Write user data (name, age) to a CSV file, then read it back and display it.
## Solution : 
- In main.py
- "csv" module is used for reading and writing csv data
- csv.writter() and csv.reader() function are used
## Notes : 
- In this challenge, we used csv module which is in-built in python for handling csv files
- To write the data in a csv file , we create a list containg the data
- Using csv.writter() the data is written into the csv file
- When we want to print or display our data, csv.reader() is used
